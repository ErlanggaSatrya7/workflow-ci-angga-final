3. Deskripsi Workflow CI:
Workflow CI ini diimplementasikan menggunakan layanan GitHub Actions. Pipeline diatur agar secara otomatis tereksekusi setiap kali ada perubahan kode (push) ke dalam repository.

4. Tahapan (Jobs) yang Dijalankan:
Pipeline CI ini menjalankan langkah-langkah otomatisasi berikut:
- Melakukan proses checkout kode dari repository.
- Melakukan setup environment Python.
- Menginstal dependensi dan library yang dibutuhkan sesuai dengan list di file requirements.txt.
